# Introduction

This is my personal website. It is hosted on vercel. It is built with astro. 

## Code to run to make changes

### Setup

1. Clone the respository: `git clone ...`
2. Install dependencies: `npm install`
3. Run local dev server: `npm run dev`

### Once you are done making changes

1. Add all changes: `git add .`
2. Commit all changes: `git commit -m "Message..."`
3. Push all changes: `git push`

To change branch on git: `git checkout "branch name"`
